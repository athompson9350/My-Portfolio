# Suraj G Rao - Portfolio

A modern, responsive portfolio website showcasing my work as an AI/ML Engineer and Software Developer.

## Features

- **Modern UI/UX**: Clean, professional design with glassmorphism effects
- **Responsive**: Fully responsive across all devices
- **Interactive Elements**: Smooth animations and transitions using Framer Motion
- **Particle Effects**: Dynamic canvas-based particle system
- **Visitor Counter**: Real-time visitor tracking with confetti celebration
- **Sections**:
  - Hero with animated typing effect
  - About with statistics
  - Skills with categorized display
  - Experience with company logos
  - Projects with featured highlights
  - Achievements with certificates
  - Leadership roles with organization logos
  - Education with institution logos
  - Certifications with verification links
  - Contact form

## Tech Stack

### Frontend
- **React 18** - UI framework
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library

### Libraries
- `canvas-confetti` - Celebration effects
- `react-countup` - Animated number counters
- `react-scroll` - Smooth scrolling
- `react-type-animation` - Typing animation
- `react-icons` - Icon library
- `@tsparticles/react` - Particle effects

### Development
- **TypeScript** - Type safety
- **ESLint** - Code linting

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Suraj-G-Rao/My-Portfolio.git
cd My-Portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview production build:
```bash
npm run preview
```

## Customization

To personalize this portfolio for your own use:

1. **Update Portfolio Data**: Edit `src/data/portfolio.js` to update:
   - Personal information
   - Skills
   - Experience
   - Projects
   - Achievements
   - Leadership roles
   - Education
   - Certifications

2. **Replace Images**: Update images in:
   - `public/logos/` - Company/institution logos
   - `public/photos/` - Certificates and achievements
   - `src/assets/profile.jpg` - Profile picture

3. **Update Links**: Change social media and contact links in `src/data/portfolio.js`
